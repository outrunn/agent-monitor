─────────────────────────────────────────────────────────────────

  🟠 Orange (40-70%) = In progress
  🔵 Blue   (<40%)  = Just started

Agent Status:
  🟢 Green circle = Running
  🔴 Red circle   = Failed (needs attention!)
  🔵 Blue circle  = Completed

Issue Highlighting:
  Light green bg = Agent assigned
  Color badges   = Agent status
  Purple tags    = Issue numbers
