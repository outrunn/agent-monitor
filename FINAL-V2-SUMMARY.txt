HOW TO USE:
─────────────────────────────────────────────────────────────────

Quick Start:
  $ cd ~/Dev/tools/agent-monitor
  $ ./open-in-xcode.sh
  (Click Play ▶️)

First Time Setup:
  1. Install GitHub CLI: brew install gh
  2. Authenticate: gh auth login
  3. Launch app (it will auto-load GitHub data)

Dashboard Navigation:
  • Left panel: Click milestone names to expand/collapse
  • Issue rows: Click agent badges to view logs
  • Right panel: Click agents for quick log access
  • Header: Toggle Dashboard/Agents view modes
  • Refresh button: Update all data instantly
